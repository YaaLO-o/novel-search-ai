## 1. 一级分类（维度）

| 维度 | 说明 | 示例 |
|------|------|------|
| **CP类型** | 感情线/配对类型 | 言情、纯爱、百合、无CP |
| **题材** | 故事背景/世界观 | 都市、古代、末世、修仙 |
| **人设** | 角色设定 | 老实人、霸总、病娇、清冷 |
| **剧情元素** | 核心情节标签 | 甜宠、虐恋、先婚后爱、破镜重圆 |
| **文风** | 叙事风格 | 治愈、沙雕、悬疑、现实向 |
| **结局** | 结局类型 | HE、BE、OE |
| **平台** | 来源平台 | 晋江、起点、番茄、长佩 |
| **状态** | 连载/完结 | 完结、连载、暂停 |

## 3. 维度关系

```
一级维度（8个）
├── CP类型
│   └── BG / BL / GL / 无CP
├── 题材
│   └── 都市 / 古代 / 末世 / ...
├── 人设
│   └── 老实人 / 霸总 / 病娇 / ...
├── 剧情元素
│   └── 甜宠 / 虐恋 / 先婚后爱 / ...
├── 文风
│   └── 治愈 / 沙雕 / 悬疑 / ...
├── 结局
│   └── HE / BE / OE
├── 平台
│   └── 晋江 / 起点 / 番茄 / ...
└── 状态
    └── 完结 / 连载 / 暂停
```

---

## 4. 使用规则

1. **每本书必须有至少一个一级维度的标签**
2. **同维度标签可以多选**（如：人设 = 老实人 + 忠犬）
3. **互斥标签不能同时出现**（如：HE + BE）
4. **维度优先级**：CP类型 > 题材 > 人设 > 剧情元素 > 其他
