# Tag 命名规范

**Version**: V0.2
**Date**: 2026-07-03

---

## 1. 基本规则

| 规则 | 说明 | 正确 | 错误 |
|------|------|------|------|
| 简体中文 | 统一使用简体中文 | 甜宠 | 甜蜜/寵愛 |
| 长度限制 | 2-8 个字 | 老实人 | 一个非常非常非常长的标签 |
| 不加符号 | 不加书名号、引号 | 霸总 | 《霸总》/ "霸总" |
| 不用缩写 | 除非广泛通用 | Happy Ending | HE（可作为别名） |
| 不重叠 | 同类型下标签不重叠 | - | - |

---

## 2. 格式要求

### 2.1 字符规范
- 使用简体中文（国家通用规范汉字）
- 数字使用阿拉伯数字（如：三生三世）
- 英文仅在通用缩写时使用（如：BG、BL、HE）

### 2.2 禁止内容
- 书名号：《》
- 引号："" ''
- 括号：（）除非是标签名的一部分
- 特殊符号：！？。，等

### 2.3 推荐写法
| 场景 | 推荐 | 不推荐 |
|------|------|--------|
| CP类型 | BG | 男女向 |
| 人设 | 老实人 | 木讷老实的人 |
| 剧情 | 先婚后爱 | 先结婚后恋爱的故事 |
| 文风 | 治愈系 | 温馨治愈风格 |

---

## 3. hashtag 格式

从推书文本自动提取时，支持 `#` 前缀的 hashtag 格式：

```
输入：#甜宠# #先婚后爱#
提取：甜宠、先婚后爱
```

**规则**：
- `#` 前缀可选（自动去除）
- 标签内不能包含 `#`
- 连续 `#tag#` 格式自动解析

---

## 4. 大小写处理

| 原始输入 | 标准化输出 |
|----------|------------|
| bg | BG |
| Bl | BL |
| he | HE |
| 甜宠 | 甜宠 |

**规则**：英文标签统一转为大写

---

## 5. 去重规则

| 输入 | 标准化 | 原因 |
|------|--------|------|
| 甜宠 / 甜蜜宠文 | 甜宠 | 合并近义词 |
| 都市 / 现代都市 | 都市 | 统一为简短形式 |
| 完结 / 已完结 | 完结 | 统一为标准形式 |

**去重策略**：
1. 预定义同义词表
2. 相似度 > 0.8 时自动合并
3. 人工审核高相似度标签
