# MVP 定义

## 当前已完成 (v0.1.0)

### 搜索功能
- [x] 关键词搜索（书名、作者、平台、简介、标签）
- [x] 平台筛选
- [x] 状态筛选（想看/在看/已看/避雷）
- [x] 三种排序（匹配度/评分/最近加入）
- [x] 标签云快捷搜索

### 书目管理
- [x] 手动录入书目
- [x] 编辑书目信息
- [x] 标签分类展示（官方/个人/社区）
- [x] 正版链接跳转

### 批量导入
- [x] 粘贴文本解析《书名》格式
- [x] 自动提取作者、平台、标签
- [x] 重复导入检测（SHA-256 哈希）
- [x] 自动建立同类推荐关系

### 数据库
- [x] SQLite 本地存储
- [x] 6 张核心表（books, tags, book_tags, import_batches, import_books, recommend_edges）
- [x] UPSERT 防重复

## 验收标准
- 搜索响应时间 < 100ms
- 支持 1000+ 书目无卡顿
- 导入 100 本书 < 5s
- 数据库文件 < 50MB
