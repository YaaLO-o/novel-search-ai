# 产品需求文档 (PRD)

## 产品名称
书荒雷达 (Novel Search AI)

## 产品定位
个人小说搜索与书库管理工具，解决"书荒"问题——帮用户快速找到想看的小说。

## 目标用户
- 个人用户（自托管，本地部署）
- 网文读者，需要管理多个平台的书单

## 核心功能

### 1. 小说搜索
- 支持按书名、作者、平台、简介全文搜索
- 标签匹配加权排序（官方标签 > 个人标签 > 社区标签）
- 三种排序：匹配度、评分、最近加入

### 2. 书目管理
- 手动录入：书名、作者、平台、简介、标签、状态
- 阅读状态：想看 / 在看 / 已看 / 避雷
- 正版链接跳转

### 3. 批量导入
- 粘贴 Obsidian 推书文本，自动解析《书名》格式
- 支持提取作者、平台、标签、简介
- 同一段落内的书自动建立"同类推荐"关系

### 4. 标签系统
- 官方标签（平台自带）
- 个人标签（用户自定义）
- 社区高频词（从推书文本提取）
- 标签云展示常用标签

### 5. 同类推荐
- 基于推荐边（recommend_edges）的关联推荐
- 共同出现在同一批推书中的书建立推荐关系

## 技术架构
- **前端**：原生 HTML/CSS/JS，无框架依赖
- **后端**：Node.js HTTP 服务器
- **数据库**：SQLite（node:sqlite）
- **部署**：本地运行，端口 5173

## 非目标
- 多用户系统
- 云端同步
- 用户认证/授权
- 移动端 App
